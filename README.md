Мой интернет-блог  
<br/>Простой блог на PHP с регистрацией пользователей, постами, комментариями и загрузкой изображений.  
<br/>Содержание  
<br/>· Структура проекта  
· Функционал  
· Использование  
· Технологии  
<br/>Структура проекта  
blog-project/  
├── index.php              # Главная страница  
├── login.php              # Страница входа  
├── register.php           # Страница регистрации  
├── create.php             # Создание поста  
├── post.php               # Просмотр поста и комментарии      
├── logout.php             # Выход из системы  
├── data/                  # Хранение данных  
│   ├── users.json  
│   ├── posts.json  
│   └── comments.json  
├── includes/         #Вспомогательные файлы  
│   ├── bootstrap.php  
│   ├── functions.php  
│   └── User.php  
├── css/  
│   └── style.css         # Стили  
└── uploads/              #Загруженные изображения  
<br/><br/>Функционал  
<br/>Регистрация и авторизация  
<br/>· Регистрация   
· Хеширование паролей  
· Сессии для авторизации  
<br/>Посты  
<br/>· Создание постов с заголовком и текстом  
· Загрузка изображений (JPG, PNG, GIF, до 2MB)  
· Просмотр отдельных постов  
· Список всех постов на главной  
<br/>Комментарии  
<br/>· Комментирование постов  
· Только для авторизованных пользователей  
· Сортировка по дате (новые сверху)  
<br/>Использование  
<br/>Для пользователей  
<br/>1\. Зарегистрируйтесь на сайте  
2\. Войдите в систему  
3\. Создавайте посты с изображениями  
4\. Комментируйте посты других пользователей  
5\. Выходите из системы при завершении работы  
<br/>Форматы данных  
<br/>· Изображения: JPG, PNG, GIF  
· Максимальный размер: 2MB  
· Текст: поддерживает HTML-разметку (с экранированием)  
<br/>Технологии  
<br/>· PHP - серверная логика  
· HTML/CSS - разметка и стили  
· JSON - хранение данных  
<br/>Безопасность  
<br/>· Хеширование паролей   
· Валидация загружаемых файлов  
· Проверка авторизации на защищенных страницах
